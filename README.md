# Varios 💰 — Control de Gastos Compartido

App PWA de control de gastos semanales con sincronización en tiempo real via Google Sheets.

## Instalación en celular

Abrí `https://[tu-usuario].github.io/varios-app/` en Chrome (Android) o Safari (iOS).

- **Android:** Chrome muestra un banner "Instalar app" automáticamente
- **iOS:** Compartir → Añadir a pantalla de inicio
